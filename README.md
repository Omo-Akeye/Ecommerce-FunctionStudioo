# E-Commerce Function Landing Page

A premium, interactive e-commerce landing page built with React, TypeScript, and Vite. The design features a dynamic split-screen room layout, interactive product hitboxes, and organic vector lighting overlays.

## Key Features


- **Dynamic Split-Screen**: The screen dividing seam adjusts automatically on window resize to optimize background image framing, and features a manual split toggle.
- **Grain Effect**: A customized canvas-based soft-light noise texture overlay for high-fidelity aesthetics.


### Installation
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```
